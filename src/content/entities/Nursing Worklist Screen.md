# Nursing Worklist Screen

หน้าจอรายการคำสั่งแพทย์สำหรับพยาบาล [[IPD]]

## Access

Inpatient → Nursing Worklist

## Features

### Search & Filter
- เลือก **Ward** หรือค้นหาจาก **Search Patients** → SEARCH

### Group By (6 แบบ)

| Group | Description |
|-------|-------------|
| **Department** | จัดเรียงตามแผนก |
| **Patient** | จัดเรียงตามชื่อผู้ป่วย |
| **Careprovider** | จัดเรียงตามแพทย์ |
| **Order Category Type** | จัดเรียงตามประเภทคำสั่ง |
| **Status** | จัดเรียงตามสถานะ |
| **Date** | จัดเรียงตามวันที่ |

### Actions

- **Execute Order**: เลือก order → Update → Execute Order → SAVE
- **IPD Consults**: ดูรายการส่งปรึกษา → Assign Careprovider (Modify)
- **Follow up**: รับ order → Execute → ทำนัดหมาย
