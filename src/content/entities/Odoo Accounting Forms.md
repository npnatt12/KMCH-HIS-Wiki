# Odoo Accounting Forms

## Purpose

Standard accounting documents, forms, and reports delivered in [Odoo ERP](/modules/odoo-erp/).

## Standard Forms

| Form | Description |
|------|-------------|
| Journal Voucher | ใบสำคัญรายวันทั่วไป |
| Payment Voucher | ใบสำคัญจ่าย |
| Receive Voucher | ใบสำคัญรับ |
| Receipt | ใบเสร็จรับเงิน |
| Purchase Order | ใบสั่งซื้อ |
| Sales Invoice | ใบแจ้งหนี้ |
| Credit Note | ใบลดหนี้ |
| Debit Note | ใบเพิ่มหนี้ |

## Reports

| Report | Use |
|--------|-----|
| Purchase Tax Report | รายงานภาษีซื้อ |
| Sale Tax Report | รายงานภาษีขาย |
| Withholding Tax PND.3 / PND.53 | ภาษีหัก ณ ที่จ่าย |
| Aged Receivable | ลูกหนี้คงค้าง |
| Aged Payable | เจ้าหนี้คงค้าง |
| Invoice Analysis | วิเคราะห์ invoice |
| Profit and Loss | งบกำไร/ขาดทุน |
| Balance Sheet | งบฐานะการเงิน |
| Partner Ledger | บัญชีแยกตามคู่ค้า |
| Cash Flow Statement | งบกระแสเงินสด |
| General Ledger | บัญชีแยกประเภท |
| Trial Balance | งบทดลอง |
